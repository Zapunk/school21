## Установка и базовая настройка сервиса SSHD

- `sudo apt install openssh-server`

- `sudo systemctl enable ssh`

- `sudo nano /etc/ssh/sshd_config`
меняем порт с 22 по умолчанию, на 2022 и разкомичиваем.

- ps - инструмент для мониторинга процессов. Эта команда позволяет выводить информацию о запущенных процессах, узнать их идентификаторы(PID), статус, потребление ресурсов и другие детали. Ключ `-e` нужен чтобы показать все запущенные процессы, а команда `grep` для того, что бы вывести только нужный нам процесс.

![Ps](img/ps.png)

- `sudo reboot`

- ключ `-tan` с командой `netstat` - это комбинация из трех ключей:
   - `-t` отображать только TCP-подключения.
   - `-a` показывает все соединения, включая ожидающие.
   - `-n` выводит IP-адреса и порты в числовом формате.

![Netstat](img/netstat.png)

На скриншоте в столбцах указаны:
   - первый: протокол
   - второй: полученые пакеты
   - третий: отправленые пакеты
   - четвертый: локальный адрес и порт
   - пятый: удаленый адрес и порт
   - шестой: состояние соединения(LISTEN - ожидают подключения, TIME WAIT - завершающиеся соединения)
   - 0.0.0.0 "слушает все интерфейсы"
